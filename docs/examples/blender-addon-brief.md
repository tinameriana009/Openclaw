# Example brief: Blender add-on for scene cleanup

## Goal

Create a Blender add-on that helps artists clean a scene before export by:

- finding empty materials
- detecting duplicate material names
- listing unapplied transforms
- offering a simple panel with batch actions

## Constraints

- target Blender 4.x
- Python only
- keep package layout minimal
- assume manual testing inside Blender

## Good first request

```text
Design a minimal Blender add-on for scene cleanup. Propose the package tree, operator list, UI panel layout, and the first implementation steps. Call out manual validation steps and any Blender API assumptions.
```
